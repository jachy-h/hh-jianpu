# 笛子移调功能产品设计

> 版本：v0.2.0  
> 创建日期：2026-03-04

## 1. 需求背景

### 1.1 用户痛点

吹笛子的用户只会使用筒音作5的指法，但谱子里有低音3（以及更低的音），无法演奏。

用户不懂什么是C调、G调，只知道要让谱子变成可以用筒音作5吹的版本。

### 1.2 需求概述

在编辑区域增加"功能区"，提供移调功能：
- 用户可选择目标调号（手动选择）
- 提供"适配筒音作5"快捷功能（自动计算所需移调半音数）
- 确认前提示用户会生成新谱，不会修改原谱

---

## 2. 功能设计

### 2.1 UI 布局

**位置**：简谱输入区域右上角，"自动保存"文字左侧

**功能区按钮**：
- 移调按钮（图标 + "移调"文字）

### 2.2 弹框设计

```
┌─────────────────────────────────────┐
│  移调设置                    ✕       │
├─────────────────────────────────────┤
│  📝 当前谱面信息：                    │
│     调号：C                          │
│     最低音：3, (低音3)               │
│                                     │
│  🎯 目标设置：                       │
│     ┌─────────────────────────┐     │
│     │ 目标调号（手动选择）      │     │
│     │ C  D  E  F  G  A  B    │     │
│     │ #  b                     │     │
│     └─────────────────────────┘     │
│                                     │
│     ───── 或者 ─────                │
│                                     │
│     🪈 适配筒                  音作5 │
│     [自动计算：需上升 2 半音]        │
│                                     │
│     适配筒音作2                     │
│     适配筒音作6                     │
│     适配筒音作1                     │
│                                     │
│  ⚠️  此操作将生成新的简谱源码，       │
│     不会修改原谱。                   │
│                                     │
│     [ 取消 ]        [ 确认移调 ]    │
└─────────────────────────────────────┘
```

### 2.3 交互流程

1. 用户点击"移调"按钮 → 弹出移调设置弹框
2. 弹框自动分析当前谱面，显示调号、最低音等信息
3. 用户选择：
   - 手动选择目标调号
   - 或点击"适配筒音作X"快捷按钮
4. 确认前显示提示："此操作将生成新的简谱源码，不会修改原谱"
5. 点击"确认移调" → 生成新的简谱源码，替换编辑器内容

## 3. 技术方案

### 3.1 笛子指法与音域

| 指法 | 最低可吹奏音 | 最低音 MIDI |
|------|-------------|-------------|
| 筒音作5 | 5, | 48 (C3) |
| 筒音作2 | 2, | 50 (D3) |
| 筒音作1 | 1, | 52 (E3) |
| 筒音作6 | 6, | 53 (F3) |

### 3.2 移调算法

1. 解析原谱最低音和最高音
2. 如果选择"适配筒音作X"：
   - 计算 `偏移 = 指法最低音 - 原谱最低音`
   - 如果偏移 > 0，整体升高 N 个半音
   - 如果偏移 <= 0，提示"无需移调，原谱已适合此指法"
3. 如果选择目标调号：
   - 计算目标调与原调的距离（半音数）
   - 整体移调

### 3.3 简谱语法转换规则

| 原音符 | 移调+1 | 移调+2 | 移调+3 | 移调+4 | 移调+5 | 移调+6 | 移调+7 |
|--------|--------|--------|--------|--------|--------|--------|--------|
| 1 | 1# | 2 | 2# | 3 | 4 | 4# | 5 |
| 2 | 3 | 3# | 4 | 4# | 5 | 6 | 6# |
| 3 | 4 | 5 | 5# | 6 | 6# | 7 | 1' |
| 4 | 5 | 5# | 6 | 6# | 7 | 1' | 1'# |
| 5 | 6 | 6# | 7 | 1' | 1'# | 2' | 2'# |
| 6 | 7 | 1' | 1'# | 2' | 2'# | 3' | 4' |
| 7 | 1' | 1'# | 2' | 2'# | 3' | 4' | 4'# |
| 1, | 2, | 2#, | 3, | 4, | 4#, | 5, | 5#, |
| 2, | 3, | 4, | 4#, | 5, | 5#, | 6, | 7, |
| 3, | 4, | 5, | 5#, | 6, | 6#, | 7, | 1' |

需要处理：
- 八度标记变化（`,` 减、'`' 增）
- 临时升降号的传递
- 调号变化（`调号：C` → `调号：D`）

## 5. 边界情况处理

1. **最高音超限**：如果原谱最高音超过指法最高音（如筒音作5最高6，原谱有7）
   - 提示用户"原谱有高音超出范围，无法适配，建议选择其他指法"

2. **无需移调**：如果原谱最低音 >= 指法最低音
   - 提示"原谱已适合此指法，无需移调"

3. **箫的支持**：箫和笛子共用同样的指法（筒音作1/2/5/6），先按笛子实现
